My Projects

My Projects

ASP-BookStore

The bookstore project employs ASP (Active Server Pages) to develop an interactive online platform for browsing and purchasing books.


Leveraging ASP's server-side scripting capabilities, the website dynamically generates pages, enabling smooth navigation and secure transactions.

Documentation accompanying the project provides valuable insights into ASP implementation, enhancing understanding of its role in facilitating dynamic web content.

JavaScript-Friends

In this project, JavaScript is utilized to manage a list of friends, offering functionalities for adding, displaying, and searching for specific names. Upon loading the page, the list of friends is rendered dynamically, enabling users to view all existing entries.


Additionally, a search feature is implemented, allowing users to input a name and instantly filter the displayed list to find matching entries. Moreover, a dedicated button is provided to trigger the search operation, enhancing user experience and interaction with the friend list.

JavaScript-Famous Car

I developed a web application showcasing famous JavaScript cars, creating detailed cards for each vehicle and incorporating a search feature for quick navigation. Routing was implemented to enable seamless transitions between the content list and individual car details, enhancing user experience and efficiency.


Additionally, lazy loading was employed for the content detail component to optimize performance, ensuring smooth navigation and faster page loading times.
